# Smart Parking Allocation System (AI Project)

## Overview
This project implements an AI-based Smart Parking Allocation System using multiple search algorithms including BFS, DFS, Greedy Best-First Search, and A* Search.

## Algorithms Used
- Breadth-First Search (BFS)
- Depth-First Search (DFS)
- Greedy Best-First Search
- A* Search Algorithm

## Heuristic
Manhattan Distance

## Features
- Grid-based parking simulation
- Multiple goal (parking) states
- Path visualization
- Performance comparison between algorithms
